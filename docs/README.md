# Documentation for Data Analysts

This folder contains all documentation needed to work with the Better Signals data infrastructure.

## Quick Start

1. **Setup**: Read `RUNBOOK_VSCODE_BIGQUERY_SETUP.md` first
2. **Data Reference**: Check `TABLES_REFERENCE.md` for available tables
3. **Common Queries**: Use `COMMON_QUERIES.md` for frequently used SQL

## Contents

| File | Description |
|------|-------------|
| `RUNBOOK_VSCODE_BIGQUERY_SETUP.md` | How to set up VSCode + BigQuery for analysts |
| `RUNBOOK_AIRBYTE.md` | Airbyte configuration and management |
| `TABLES_REFERENCE.md` | All BigQuery tables with schemas |
| `DATA_DICTIONARY.md` | Field definitions and business logic |
| `COMMON_QUERIES.md` | Frequently used SQL queries |
